Data Sources

Primary Data Source: IGDB API

Our primary game data is sourced from the Internet Game Database (IGDB) API. IGDB is a comprehensive database for video games, offering a wide range of information including game titles, storylines, summaries, ratings, genres, game modes, and more.

We periodically retrieve and process data from the IGDB API to ensure our database remains updated with accurate information. All data displayed on this website is subject to IGDB's terms of service and licensing.

Additional Information

While IGDB serves as our main data provider, supplementary information and visual assets (e.g., screenshots, where applicable) may be curated or linked from other publicly available sources to enhance the user experience. All third-party content is used under fair use principles or with appropriate attribution.

This website is a personal portfolio project and is not affiliated with IGDB or any game publishers directly. It is intended for educational and demonstration purposes only.
